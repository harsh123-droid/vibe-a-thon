# 🎮 Level Up Portfolio – Harsh Jain

## 🚀 Overview

This project is a **gamified interactive portfolio** built for the Vibe-a-thon hackathon.

Instead of a traditional resume, this website turns a portfolio into an **interactive game-like experience**, where users gain XP, unlock achievements, and explore sections as levels.

---

## ✨ Features

* 🎯 Scroll-based XP system with dynamic level progression
* 🏆 Achievement unlock system (section-based)
* 🌈 Cursor-based blended neon blob effect
* 🧲 Magnetic hover interactions
* 🎴 Interactive tilt project cards
* 🌗 Animated dark/light mode toggle
* 📱 Fully responsive design

---

## 🧩 Components Implemented

* ✅ Hover Effect → Tilt cards & magnetic buttons
